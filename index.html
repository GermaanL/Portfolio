Son 4 modulos CPU, MEMORIA, FILESYSTEM y KERNEL que se comunican mediante sockets, cumpliendo un protocolo de comunicación.

Trabajo Práctico - Java - HTML - CSS - Bootstrap - MySQL - Javalin - Hibernate

En este trabajo se tuvo que realizar una desarrollo web de manera grupal (un equipo de 3) para una ONG, siguiendo los requerimientos indicados por la cátedra a lo largo del año. Se utilizó JAVA como lenguaje de programación del Backend con Javalin como framework, Hibernate como ORM y MySQL como sistema de gestión de bases de datos; HTML, CSS, Bootstrap, y JavaScript para el Frontend. Se utilizaron distintas tecnologías e integraciones para los requerimientos como Datadog para monitoreo de recursos, SpotBugs para analisis de vulnerabilidad de código estático y brokers con HiveMQ para comunicacion entre componentes
